# モナカの中 建築事務所（架空サイト）

架空の建築設計事務所「モナカの中 建築事務所」の公式サイトを想定して制作した、  
静的Webサイト（HTML / CSS / JavaScript）です。

2021年頃に、Web制作の学習およびポートフォリオ目的で作成しました。

## サイト概要

- 建築設計事務所のコーポレートサイトを想定
- トップページのみのシングルページ構成
- ビジュアル重視のレイアウトとアニメーション表現を意識
- ニュース表示、モーダル表示、アコーディオンUIを実装

※ 本サイトに掲載されている会社名・実績・人物・連絡先等はすべて架空の内容です。

---

## 使用技術

- HTML5
- CSS3  
  - レスポンシブ対応（スマートフォン / タブレット）
  - Grid / Flexbox
  - CSSアニメーション
- JavaScript（Vanilla JS）
  - 文字アニメーション
  - モーダルウィンドウ
  - ニュース自動切り替え
  - アコーディオンメニュー

---

## 実装・工夫ポイント

- 動画背景（MV）を使用し、建築事務所らしい世界観を演出
- CSS Gridを用いた大胆なレイアウト設計
- JavaScriptによる
  - 見出しテキストのアニメーション表示 
  - Newsセクションの自動切り替え 
  - モーダルによる詳細表示 
- スマートフォンとタブレットでレイアウトを切り替え、可読性を確保
